# Curso_IA_SENAI

Repositório com os códigos e projetos desenvolvidos durante o curso de **Inteligência Artificial aplicada à programação**, no SENAI.

## Sobre o projeto

O objetivo deste repositório é organizar as atividades feitas ao longo das aulas, explorando diferentes usos de IA com Python, automação, visão computacional, banco de dados e integração com hardware.

## Conteúdos abordados

* Agentes de IA;
* Uso da API da OpenAI;
* Automação com Python;
* Visão computacional com webcam;
* Leitura e interpretação de imagens;
* Geração e transcrição de áudio;
* Integração com ESP32;
* Comunicação serial;
* Banco de dados com SQLite;
* Uso de documentos como base de conhecimento.

## Observações

Alguns códigos precisam de arquivos locais, webcam, chave da OpenAI ou ESP32 conectado para funcionar corretamente.

Este repositório tem finalidade educacional e foi desenvolvido durante as aulas do curso.
